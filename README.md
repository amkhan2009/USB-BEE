The build is quite simple, it is compromised of 5 USB ports
-One Upstream USB C port (connects from your PC)
-4 Downstream USB A ports (Connects to misc devices)

It adds more ports and allows for you computer to be able to gain more USB slots without needing to buy a entirely new PC.
The board is quite straight forward with a rectangular shape and two holes in the corner for future CAD building of a case or a keychain.
